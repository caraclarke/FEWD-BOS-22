Programming Tools

We will be using the following tools (to start):

  • Github: code repository
  • Google Chrome: web browser
  • Sublime Text: text editor
  • Codepen: real time text editor
  • w3schools, StackOverflow, Google: resources

Github:

  • Stores files like Dropbox or Google Drive, but stores code.
  • Stores a history of files and the changes that happen within each document.
  • Hosts files on the cloud so you can share the finished product with other people.
  • Designed to allow for multiple engineers to work on the same project.
